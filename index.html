Chọn ngân hàng từ danh sách gốc VietQR để tự điền mã/BIN/logo. "Tên gợi nhớ" dùng để gọi tài khoản qua link API (vd: ?bank=Vi+tien+mat). Muốn đặt tài khoản mặc định thì đóng bảng này rồi dùng nút ☆ Đặt làm mặc định ở form tạo QR. Sửa xong bấm Lưu lên GitHub để ghi lại data/my-accounts.json. Danh sách ngân hàng gốc (data/vietqr-banks.json) chỉ để tra cứu, không cần đồng bộ — nếu chưa có file này thì trang tự lấy trực tiếp từ VietQR, hoặc bấm Làm mới ngân hàng từ VietQR ở trên.

Tải ảnh QR
Cú pháp gọi qua link API

?amount=100000&addInfo=... — dùng luôn tài khoản mặc định, trả thẳng ảnh QR.
?bank=<tên gợi nhớ>&amount=100000&addInfo=... — chọn tài khoản theo tên gợi nhớ đã đặt ở tab Danh sách.
?bank=VCB&stk=1031451081&amount=100000&content=...&redirect=1 — kiểu cũ, chỉ định thẳng mã ngân hàng + số TK.
Thêm &format=text để trang chỉ in ra link ảnh thay vì chuyển thẳng. Nút Sao chép link API ở trên tự tạo link theo tài khoản/nội dung đang xem.
